## Installation and Use
```
# Clone the repository
git clone https://github.com/thomaztorres/proffy-study

# Enter the directory
cd proffy-discovery

# Download dependencies
npm i

# Run the server
npm run dev
```
After this, acess you browser and type http://localhost:5500/
